# shopping_list

Pagina web mobile che virtualizza una lista della spesa.

Tecnologie utilizzate:

- HTML5;
- CSS3;
- PHP7.3;
- JS6 + JQuery 3.3.1;
- MySQL.

Cliccando il bottone Checked(v) viene eliminato dalla lista (e dal db) la riga a cui si riferisce il bottone.
